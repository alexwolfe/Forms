Getting Started

Setup Forms

  1. Download Forms and add files to your website.
  2. Add css in the head of your webpage. 
    <link rel="stylesheet" href="css/forms.css">
<script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>
<script type="text/javascript" src="js/forms.js"></script>
  3. Create forms in your html.

Customize Forms

  1. Fork or Download the Forms project on github.
  2. Make sure you have Sass and Compass installed.
  3. Edit the _options.scss with your own custom values
  4. In the root of the Forms directory type compass watch on the command line.
  5. The forms.css file should now be updated
